# NEXUS'26

A modern full stack event management platform built using React, TypeScript, Firebase, Zustand and TailwindCSS.

## Features

- Google Authentication
- Student Pass Generation
- Merchandise Store
- Shopping Cart
- Firestore Integration
- Dashboard
- Responsive UI
- Modern Animations

## Tech Stack

Frontend:
- React
- TypeScript
- Vite
- TailwindCSS
- Framer Motion

Backend:
- Firebase Authentication
- Firestore Database

State Management:
- Zustand

## Installation

npm install

npm run dev
